# User Stories — Fluxo CT-e

> Gerado pelo Redator em 2026-06-08

## Épico: Emissão de Documento Fiscal (CT-e)

**US-01: Transmissão SEFAZ**
Como faturista operacional,
Eu quero preencher os dados do frete e transmitir para a SEFAZ,
Para obter o protocolo de autorização e poder circular com a carga legalmente.

**US-02: Carta de Correção**
Como faturista operacional,
Eu quero emitir uma Carta de Correção (CC-e) sobre um CT-e já emitido,
Para corrigir erros de digitação (ex: endereço, observações) sem precisar cancelar e refazer o documento.

**US-03: Inutilização**
Como gerente fiscal,
Eu quero inutilizar uma faixa de numeração de CT-e que não foi e não será usada,
Para evitar autuações por salto de numeração na auditoria da Receita.

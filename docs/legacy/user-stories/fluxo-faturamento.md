# User Stories — Fluxo Faturamento

> Gerado pelo Redator em 2026-06-08

## Épico: Faturamento e Recebimento

**US-01: Agrupamento de Conhecimentos em Fatura**
Como auxiliar de faturamento,
Eu quero selecionar múltiplos CT-es emitidos "A Prazo" para um mesmo sacado e agrupá-los em uma Fatura,
Para emitir uma cobrança única agrupada para o cliente.

**US-02: Geração de Remessa CNAB**
Como auxiliar de faturamento,
Eu quero gerar um arquivo de Remessa no padrão do banco credenciado,
Para registrar os boletos e garantir a cobrança oficial.

**US-03: Processamento de Retorno Automático**
Como tesoureiro,
Eu quero importar o arquivo de Retorno do banco,
Para que o sistema identifique quais faturas foram pagas e faça a baixa automática, creditando o caixa sem esforço manual.
